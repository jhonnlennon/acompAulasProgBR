Vamos conhecer a propriedade Display!
Nos temos:
  • display:none
  • display:block
  • display:inline
  • display:inline-block




Display Block

O display block é um bloco e vai do inicio até o fim da página, quando ele chega até o fim, automaticamente o proximo elemento vai para a próxima linha.
O parágrafro é um display block. No display block, podemos definir uma largura e altura especifica.
Uma DIV é um display block.



Display Inline

O display inline ele fica na mesma linha. O span é um display inline.
O display inline, NÃO da para mudar altura ou largura dela.
Um Span é tem a propriedade display inline.



Display Inline-block

O display Inline-block, ele vai manter na mesma linha, mas deixa mudar a altura e largura.
Ele vai ter as duas propriedades juntas, o block que define a altura e largura e também o inline que deixa o item na mesma linha.



Display None

Ele nunca vai mostrar o elemento na tela. Simples.
