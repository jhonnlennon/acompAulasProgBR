Position

Static: O position Static nada mais é que o padrão para todos os elementos.
Isso também significa que as propriedades TOP, BOTTOM, LEFT e RIGHT não funcionam.

Relative: O relativo vai inicialmente ser como o Static, porém, ele pode receber valores:
Right, top, left e bottom. E assim ele vai dar um distanciamento de acordo com o valor das propriedades.
Deve se entender que o Relative vai sempre contar a partir da posição static original.

Fixed: O Fixed também recebe os valores de right, top, left e bottom, porém, ele vai ter a relação com a nossa janela! Nos eixos x e y, sendo eles 0. Por exemplo, chat funciona desta forma.

Absolute: O display absolute, ele vai se posicionar a partir do elemento relative mais proximo a ele.
Caso não ocorra, ele vai pegando o mais próximo até que enfim chega no body.

Sticky: Ele define que o elemento não pode ter o valor menor do que o setado nas suas propriedades.
Por exemplo, se colocar o código "top:0", ele ficaria sempre apresentavel na tela. Ja que ele nao recebe valor negativo.
