Teoria 1

Uma das formas que temos para centralização, seria margem no estilo.
Neste caso temos que usar o Margin:Auto, que vai permitir deixar o elemento sempre equilibrado ao centro.

Teoria 2

Uma forma de alinhamento para a direita seria o Margin-Left.
Ele vai permitir o elemento sempre ficar alinhado a direita.

Alinhamento de Imagens

A primeira forma seria transformar o elemento img, em um display tipo block, que vai permitir mudar de posição.
Neste caso passa a funcionar o margin, ou o margin lef.

No caso da tag IMG, não funciona o inline-block. Para o uso do inline-block temos que adicionar a tag IMG dentro de uma DIV por exemplo e usamos o text-align:center por exemplo.
