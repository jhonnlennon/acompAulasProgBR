Entendendo o Float

O Float é uma propriedade do css que permite deixar um elemento flutuando.
O código pode ser float:left, float:right... por padrão é float:none;
Existe uma propriedade do css que elimina a configuração do float, o código é:
clear:both, que não permite qualquer elemento flutuando, mas também temos o clear:right e o clear:left, intuitivamente não permite elementos flutuando de acordo com o lado escolhido.
