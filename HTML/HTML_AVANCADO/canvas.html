Canvas

O canvas é uma forma que temos para usar desenhos na tela.

Todo canvas, precisa ter o seu ID. A tag se inicia com [CANVAS].

Atenção!!!

É importante colocar a largura e altura, porque senão vai ficar completamente torto. E fica esticado no css. Use o ATRIBUTO!

Desenhando um Retangulo

Entendendo o beginPath() e closePath()

beginPath() é deve ser usado quando abrimos uma nova linha.

closePath() pode ser usado para fechar a linha ao encontro da origem, ou seja, volta para o começo.
