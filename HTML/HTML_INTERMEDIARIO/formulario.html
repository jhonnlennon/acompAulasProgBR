Criando um formulário!

Um formulário se inicia com a tag "< form>"

Devemos sempre dizer qual é o seu método, sendo eles:

  • GET: Os dados enviados são pelo endereço.
  • POST: É o inverso do GET.

Também devemos dizer qual é sua "ACTION", normalmente para onde deve ir o formulário.
