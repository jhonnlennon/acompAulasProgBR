Caso a gente não defina a unidade de medida da fonte, por padrão temos 16px.

Conteúdo do container
Conteúdo do inside, 1em é unidade do elemento, como essa div esta dentro da div container, ele esta herdando o tamanho da div container.

Isso significa, 1 vez o texto daquele elemento, se aumentar para 2em, fica como esta.

Ele multiplica o valor atual pelo numero que adicionar.

Também serve para o width e o height.


A unidade relativa "VW" é uma unidade relativa ao tamanho da tela que esta sendo exibida!
50vw é 50% da janela.
50vh também serve como altura.

Também temos o "vmin", segue a linha de raciocínio das porcentagens.
Mas ele é relativo ao menor elemento, se for a altura ou a largura.

Também temos o "vmax", tem a mesma linha de raciocínio quanto a porcentagem.
Mas ele modifica de acordo com a altura ou largura maxima.

Também temos a "porcentagem", a porcentagem vai ser sempre relativa a porcentagem do elemento pai.
por exemplo, se esta div for a primeira, essa div vai ter a porcentagem do body.
Se ela tiver dentro de uma outra div, ela vai sempre ter a porcentagem de acordo com a div que estive em cima dela.
